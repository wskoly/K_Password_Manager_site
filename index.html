K Password Manager

K Password Manager
v1.0.0

A desktop GUI application that can generate random strong passwords and store those along with username, system URL, and system tag in an encrypted database which requires a master password to decrypt.

Features

  • Own encrypted database.
  • Customized random password generation feature.
  • You can add,edit,view,delete data.
  • You can export and import encrypted data.
  • You can view login logs.
  • Fancy fonts in GUI.
